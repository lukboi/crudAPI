CRUD com Fetch API

Este projeto foi desenvolvido para praticar o uso da Fetch API em operações de CRUD (Create, Read, Update, Delete) com JavaScript puro. Utilizamos a API pública JSONPlaceholder para simular essas operações com posts. Cada função em JavaScript realiza uma operação específica:

  • Carregar Posts: Recupera os posts da API e exibe no navegador.
  • Criar Novo Post: Envia um novo post para a API e adiciona à lista de posts.
  • Atualizar Último Post: Atualiza o último post criado.
  • Deletar Último Post: Remove o último post criado da API e da lista.
A interface é responsiva e foi criada usando variáveis CSS para facilitar a manutenção e mudanças de cores.
